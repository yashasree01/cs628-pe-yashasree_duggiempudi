# Input
The input for this application is a list of movies defined in the React component. Each movie contains a title, genre, and release year. The user also provides input by selecting a genre from the dropdown menu and by clicking on a movie card.

# Process
The application is built using React functional components and hooks. The movie data is stored using useState. A dropdown menu is created dynamically by extracting unique genres from the movie list. When a user selects a genre, the list is filtered accordingly. If "All Genres" is selected, all movies are displayed. When a movie is clicked, an alert shows the movie title. JSX is used to render the UI and CSS is used for styling.

# Output
The output is a web page that displays a list of movies in card format. Each card shows the title, genre, and release year. The user can filter movies using the dropdown menu, and clicking on a movie displays its title in an alert box.
